## About Me

具备扎实的嵌入式 Linux 内核与设备驱动开发经验，熟悉 ARM/SoC 平台架构及常见外设接口，能够独立完成驱动设计、调试与系统集成工作。
熟练使用 C 语言进行内核态开发，理解 Linux 内核子系统框架及驱动模型，具备较强的问题定位与性能优化能力。

**核心技能：**

* 熟悉 Linux 内核驱动框架（platform、I2C、SPI、GPIO、DRM、V4L2、IRQ等）
* 熟悉 Device Tree 机制，能够独立完成 DTS/驱动适配与调试
* 熟悉 DRM/KMS、显示接口（LVDS、MIPI、DP）及视频数据通路
* 熟悉中断机制、内存管理、同步机制（spinlock、mutex、waitqueue 等）
* 熟悉 regmap、devres、of_graph 等常用内核接口与驱动辅助框架
* 熟练使用 Git、Makefile、Buildroot/Yocto 进行系统构建与版本管理
* 熟悉 GStreamer、视频采集与编解码流程
* 熟练使用示波器、逻辑分析仪等硬件调试工具

**工程能力：**

* 能够根据硬件原理图快速完成驱动框架设计与实现
* 具备复杂问题定位能力，能够通过 log、trace、寄存器分析、波形抓取等手段排查系统问题
* 具备系统级调试经验，能够定位启动、时序、带宽及稳定性问题
* 熟悉性能优化方法，包括中断优化、DMA优化、缓存优化等

**扩展能力：**

* 熟悉 FPGA / Verilog 基础，能够配合硬件工程师进行接口联调
* 熟悉图像处理数据流、缓存架构及实时视频处理系统设计
* 具备跨团队协作经验，能够参与系统架构设计与技术评审
